# Smart Bus Overcrowding & Passenger Analytics System

## Project Overview
This project is an IoT-based Smart Bus Monitoring System developed using Arduino and Tinkercad simulation platform.

The system performs:
- Passenger counting
- Occupancy monitoring
- Overload detection
- Emergency alert generation
- Automatic door locking

---

## Features
✅ Passenger Entry Counting  
✅ Passenger Exit Counting  
✅ Occupancy Percentage Monitoring  
✅ Overload Detection  
✅ Emergency LED & Buzzer Alerts  
✅ Automatic Door Locking using Servo Motor  
✅ Entry Blocking when Bus is Full  
✅ LCD-Based Live Monitoring  

---
